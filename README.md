# Physics MC Practice Exam Generator

A comprehensive, interactive multiple-choice physics practice exam application featuring 50 randomized questions across five core mechanics topics.

## Features

✅ **50 Physics Questions** organized into 5 exams:
- **Exam 1.1B**: Kinematics / Motion Basics (10 questions)
- **Exam 2.2A**: Newton's Laws / Forces / Circular Motion (30 questions across 3 versions)
- **Exam 3.1C & 3.2A**: Work / Energy / Potential Energy / Power (20 questions)
- **Exam 4.1A & 4.2A**: Momentum / Collisions / Rotation / Torque (20 questions)

✅ **Smart Question Pool Management**
- 10 random questions per test
- No repeats from the previous test
- Full cycle through all questions before reuse
- Pool progress tracking

✅ **Interactive Answer Selection**
- Click buttons A–E to select answers
- Visual feedback (selected state)
- Change answers anytime before submission

✅ **Instant Grading & Detailed Feedback**
- Score display (X / 10)
- Per-question breakdown (correct/incorrect)
- Correct answer shown for missed questions
- Educational reasoning for each answer

✅ **Responsive Design**
- Mobile-friendly layout
- Ocean Slate color palette
- Space Grotesk typography
- Smooth animations and transitions
- Accessible UI components

## Usage

1. Open `index.html` in a web browser
2. Click **Begin New Test** to generate a random 10-question exam
3. Select your answer for each question (A–E)
4. Track your progress in the status bar
5. Click **Submit Answers** to grade your test
6. Review feedback with correct answers and reasoning
7. Click **Begin New Test** again for a fresh exam

## Files

- **index.html** – Main structure and layout
- **script.js** – Question pool (50 questions), logic, grading, and state management
- **style.css** – Responsive styling with design system variables

## Questions Included

Each question includes:
- Clear, educational prompt
- Five answer choices (A–E) or two for true/false
- Correct answer key
- Detailed reasoning/explanation
- Topic classification
- Optional diagram references
